# WebCalculator

Простое веб-приложение калькулятора с CI/CD пайпланом.

## Функциональность

- Сложение, вычитание, умножение, деление
- Возведение в степень
- REST API интерфейс
- Автоматическое тестирование
- CI/CD пайплан через GitHub Actions
